Abstract

An ontology for describing boreholes, core samples, sensors, and related geological data at the Bedretto Underground Laboratory for Geosciences and Geoenergies.

Description

This ontology provides classes and properties for representing boreholes, borehole logs, core samples, sensors, and related metadata for geoscientific research. It is designed to support data integration, sharing, and documentation for the Bedretto project.

Borehole log

http://bedretto.org/ontology#BoreholeLog

Well logging, also known as borehole logging is the practice of making a detailed record (a well log) of the geologic formations penetrated by a borehole. The log may be based either on visual inspection of samples brought to the surface (geological logs) or on physical measurements made by instruments lowered into the hole(geophysical logs). Some types of geophysical well logs can be done during any phase of a well's history: drilling, completing, producing, or abandoning. Well logging is performed in boreholes drilled for the oil and gas, groundwater, mineral and geothermal exploration, as well as part of environmental and geotechnical studies., A description of a physical feature along the length of a borehole. May be continuous data (i.e. downhole geophysical) or discrete sections (i.e. geologist log)., A systematic recording of data from a well.

  • Example: Borehole logging; well logging; downhole log;

Core

http://bedretto.org/ontology#Core

A core run is defined by the drilling that happens during the time in which one core barrel is in the borehole. It starts when drilling succeeds with the empty core barrel and ends when the core barrel is full or has to be retrieved from the borehole for other reasons. The entire drill core retrieved from a borehole is the sum of drill core from all core runs in this borehole. It is also called the sampling material. IODP/ICDP naming example: 5054-1-A-178Z (Expedition number-Drill site number-Borehole character-Core run number and coring type designation)

Core sample

http://bedretto.org/ontology#CoreSample

A core sample is a cylindrical section of rock, soil, or other material that is extracted from the ground using a drilling process. It is typically used for geological, environmental, or engineering studies to analyze the composition and properties of subsurface materials.
